# 🚀 Cyberpunk HUD Vision

A futuristic browser-based **Cyberpunk HUD & X-Ray Vision Engine** built using **HTML5, JavaScript, Canvas API, and MediaPipe Holistic**.

This project combines:

- ✨ Real-time hand tracking
- ⚡ Neural particle rendering
- 🌌 Cyberpunk portal visualization
- 🧠 AI-style body scanning system
- 💥 Dynamic glitch & scan effects
- 🖐️ Multi-hand gesture interaction
- 🎨 Futuristic HUD overlays
- 🎥 Live webcam rendering

Inspired by cinematic holographic interfaces, sci-fi AI systems, and cyberpunk aesthetics.
